# L2Scan explorer

A web-based tool that allows users to analyze X Layer and other layer 2 networks. It provides a user-friendly interface for viewing transaction history, checking account balances, and tracking the status of network activity on X Layer.

Visit L2Scan explorer [here](https://zksync.l2scan.co/ "l2scan").

**Supported network**: 
- X Layer mainnet
- X Layer testnet
