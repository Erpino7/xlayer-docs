# Cross-chain

## Layer Zero

LayerZero is an interoperability protocol that connects blockchains (50+ and counting), allowing developers to build seamless omnichain applications, tokens, and experiences.

View [here](https://docs.layerzero.network/v2/developers/evm/technical-reference/endpoints "Layerzero endpoint") for the endpoint IDs and addresses required for X Layer.

**Supported network**:
- X Layer mainnet
